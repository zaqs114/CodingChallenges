# 00 Name generator

Simple credentials generator.

Names and surnames are polish.

### Instructions
Clone it and compile in IntelliJ IDEA.

### Example output
```
   Male or Female?
   male
   Name Micios
   Surname: Kolacki
   Email: micios.kolacki54@bing.com
   Password: mvvmn^!17
   ```
